## <img src="https://raw.githubusercontent.com/rosgear/rg-be-references-media_types/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Справочник «Типы медиаданных»

Справочник «Типы медиаданных» используется для назначения соответствующим элементам коллекции (справочника) требуемого типа данных.

Справочник входит в состав Панели управления GePanel и cлужит расширением для [модуля «Справочники»](https://github.com/rosgear/rg-be-reference).

### Скриншот
<img src="https://github.com/rosgear/rg-be-references-media_types/blob/main/assets/help/grid.png?raw=true">

### Установка

Для добавления справочника в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-references-media_types
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-references-media_types": "*"
}
```
или скачать архив на [странице справочника](https://rosgear.ru/component/rg-be-references-media_types/) в каталоге приложений RosGear.

После добавления справочника в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

## <img src="https://raw.githubusercontent.com/rosgear/rg-be-references-media_types/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Reference «Media types»

The «Media Types» reference is used to assign the required data type to the corresponding elements of the collection (reference).

The reference is part of the GePanel Control Panel and serves as an extension for the [«References» module](https://github.com/rosgear/rg-be-reference).

### Screenshot
<img src="https://github.com/rosgear/rg-be-references-media_types/blob/main/assets/help/grid.png?raw=true">

### Installation

To add the reference to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-references-media_types
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-references-media_types": "*"
}
```
or download the archive from the [reference page](https://rosgear.ru/component/rg-be-references-media_types/) in the RosGear application catalog.

After adding the reference to the project, install it into the web application edition using the GePanel Control Panel.
